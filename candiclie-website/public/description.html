Role Description

This is a remote internship role for a User Interface Designer. As an intern, you will be responsible for assisting with visual design, creating mockups, implementing front-end development, and working on user experience (UX) and user interface design. You will have the opportunity to contribute to real-world projects and gain hands-on experience in the field of UI design.

Qualifications

  • Strong skills in visual design, user experience (UX), and user interface design

  • Experience with creating mockups and implementing front-end development

  • Knowledge of industry-standard design tools and software

  • Ability to work independently and remotely

  • Excellent communication and collaboration skills

  • Attention to detail and a strong eye for aesthetics

  • Enrollment in or recent graduation from a relevant design or related program

  • Portfolio or examples of previous design work
